Subscription Site Redesign

Project Required /

My biggest project has been redesigning 11 subscription sites, transforming basic static pages into modern microsites.

This process included:

  • User research - what information did people really need?
  • A/B testing
  • Wordpress child theme creation, using Divi as parent theme
  • SASS for child theme files
  • GIT process
  • Custom plugins for forms and pricing tables
  • Providing training on Wordpress and Divi theme
  • GA event tracking
  • Use of HotJar to monitor user experience
  • Implementation of basic curreny detector
